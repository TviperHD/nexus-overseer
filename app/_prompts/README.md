# Prompts Folder

This folder contains prompt files used for AI-assisted development sessions.

## Files

- **`RESEARCH_PLANNING_SESSION.md`** - Prompt for research and planning sessions. Copy and paste at the beginning of planning/research sessions.
- **`AI_SESSION_START.md`** - Prompt for implementation sessions. Copy and paste at the beginning of coding/implementation sessions.
- **`AI_SESSION_END.md`** - Template for ending implementation sessions. Use to document session completion.

## Usage

1. **For Research/Planning Sessions:**
   - Copy contents of `RESEARCH_PLANNING_SESSION.md` at session start
   - Focus on research, planning, reviewing documentation
   - Do NOT modify code files

2. **For Implementation Sessions:**
   - Copy contents of `AI_SESSION_START.md` at session start
   - Focus on coding, implementation, testing
   - Follow all rules in `.cursor/rules/` folder

3. **At Session End:**
   - Use `AI_SESSION_END.md` template to document session completion
   - Create session overview in `app/session-overviews/`
   - **Note:** Session overview should have the same name as the checklist being worked on (no dates/times)
